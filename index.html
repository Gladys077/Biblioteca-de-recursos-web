Bienvenidos a la librería web Front End de objetos.

En este sitio encontrarás numerosas implementaciones para web realizadas en HTML, CSS y JavaScript. Podrás copiar y descargar el código, así como crear tus propias implementaciones directamente desde objetos. Sin más preámbulos, te invito a leer las instrucciones de uso rápido y a disfrutar de la experiencia

Bienvenido al inicio rápido:

Solo necesitas pegar el enlace en tu HTML:

<link rel="stylesheet" href="https://github.com/LionelStaricoff/bilblioteca-de-recursos-web/blob/main/css/style.css">

También puedes incluir la URL dentro del import en tu archivo style.css:

@import url('https://github.com/LionelStaricoff/bilblioteca-de-recursos-web/blob/main/css/style.css');

Para crear objetos solo debes enlazar el modulo dentro de tu archivo.JS:

import * as objetos from 'https://github.com/LionelStaricoff/bilblioteca-de-recursos-web/blob/main/js/objectos.js';
